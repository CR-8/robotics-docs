# Wireless Communication Systems

Enable remote control and data transmission between robots and operators.

## Radio Frequency (RF) Modules

### NRF24L01+

**Specifications**:
- **Frequency**: 2.4 GHz ISM band
- **Range**: 100m (outdoor), 30m (indoor)
- **Data Rate**: 250kbps - 2Mbps
- **Channels**: 125 channels
- **Cost**: $1-3

**Use Cases**: Point-to-point control, sensor networks

```cpp
#include <SPI.h>
#include <RF24.h>

RF24 radio(9, 10); // CE, CSN pins
const byte address[6] = "00001";

// Transmitter
void setup() {
  radio.begin();
  radio.openWritingPipe(address);
  radio.setPALevel(RF24_PA_MIN);
  radio.stopListening();
}

void loop() {
  const char text[] = "Hello";
  radio.write(&text, sizeof(text));
  delay(1000);
}
```

### HC-12 (433MHz)

- **Range**: Up to 1000m (outdoor)
- **Power**: 100mW
- **Interface**: UART (like Serial)
- **Use**: Long-range telemetry

## Bluetooth

### Bluetooth Classic (HC-05/HC-06)

- **Range**: 10-100m
- **Speed**: Up to 3 Mbps
- **Use**: RC cars, Arduino projects
- **Power**: Medium

```cpp
// Simple Bluetooth control
void loop() {
  if (Serial.available()) {
    char command = Serial.read();
    
    switch(command) {
      case 'F': moveForward(); break;
      case 'B': moveBackward(); break;
      case 'L': turnLeft(); break;
      case 'R': turnRight(); break;
      case 'S': stop(); break;
    }
  }
}
```

### Bluetooth Low Energy (BLE)

**Modules**: HM-10, ESP32 (built-in)

- **Power**: Very low (years on coin cell)
- **Range**: 50-100m
- **Use**: IoT, wearables, mobile app control

## WiFi

### ESP8266 / ESP32

**Advantages**:
- High bandwidth
- Internet connectivity
- Web server capabilities
- Long range

**Disadvantages**:
- Higher power consumption
- More complex setup

```cpp
#include <WiFi.h>
#include <WebServer.h>

const char* ssid = "RobotAP";
const char* password = "12345678";

WebServer server(80);

void handleForward() {
  moveForward();
  server.send(200, "text/plain", "Moving Forward");
}

void setup() {
  WiFi.softAP(ssid, password);
  
  server.on("/forward", handleForward);
  server.on("/backward", handleBackward);
  server.on("/stop", handleStop);
  
  server.begin();
}

void loop() {
  server.handleClient();
}
```

## LoRa (Long Range)

**Modules**: SX1278, RFM95W

- **Range**: 2-15km (line of sight)
- **Frequency**: 433MHz, 868MHz, 915MHz
- **Speed**: Low (0.3 - 50 kbps)
- **Power**: Very low
- **Use**: Long-range telemetry, agricultural robots

## RC Radio Systems

### Professional RC Systems

**Brands**: FlySky, FrSky, Spektrum, Futaba

- **Channels**: 4-16 channels
- **Range**: 1-2km
- **Reliability**: Very high
- **Cost**: $30-$500

**Use**: Drones, RC cars, combat robots

### PPM/PWM Signals

```cpp
#include <Servo.h>

Servo esc;

void setup() {
  esc.attach(9);
  pinMode(10, INPUT); // RC receiver channel
}

void loop() {
  int pulse = pulseIn(10, HIGH, 25000);
  
  if (pulse > 1000 && pulse < 2000) {
    esc.writeMicroseconds(pulse);
  }
}
```

## Communication Protocols

### UART (Serial)
- Simple, 2-wire
- HC-05, GPS modules

### SPI
- Fast, multiple devices
- NRF24L01+, SD cards

### I2C
- 2-wire, multiple devices
- Sensors, displays

### CAN Bus
- Industrial standard
- Automotive robotics
- Noise-resistant

## Security Considerations

⚠️ **Important**:

1. **Encryption**: Use AES for sensitive data
2. **Authentication**: Verify sender identity
3. **Frequency Hopping**: Prevent jamming
4. **Failsafe**: Auto-stop if signal lost

```cpp
// Simple failsafe
unsigned long lastSignal = 0;

void loop() {
  if (Serial.available()) {
    processCommand();
    lastSignal = millis();
  }
  
  // If no signal for 2 seconds, stop
  if (millis() - lastSignal > 2000) {
    emergencyStop();
  }
}
```

## Building a Complete Wireless System

1. **Choose Technology**: Based on range/speed needs
2. **Implement Protocol**: Define command structure
3. **Add Failsafe**: Handle connection loss
4. **Test Range**: Verify in real conditions
5. **Optimize Power**: Balance range vs battery life
